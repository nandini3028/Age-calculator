# Age Calculator
This is a responsive Age Calculator web application developed using HTML, CSS, and JavaScript.

## Features
- Select date of birth using a date picker
- Calculate exact age in years, months, and days
- User-friendly interface
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How It Works
The user selects their date of birth and clicks the "Calculate" button. The application calculates and displays the exact age in years, months, and days.
